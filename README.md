# WPF Binding and Localization Demo

Демонстрационное WPF-приложение, разработанное в рамках лабораторных работ по курсу "Объектно-ориентированное программирование". Проект иллюстрирует работу с привязками (Binding), триггерами и локализацией в WPF.

---

## Лабораторная работа №1: Привязки и триггеры

### Описание

Приложение демонстрирует различные типы привязок и триггеров в WPF с использованием паттерна MVVM. Реализовано в двух ветках:

- **master** – ручная реализация ViewModel через `INotifyPropertyChanged`
- **with-toolkit** – реализация с использованием пакета `CommunityToolkit.MVVM`

### Функциональность

Приложение содержит пять вкладок:

- **Привязка по умолчанию** – демонстрация работы Default-привязки, разница между LostFocus и PropertyChanged
- **Двухсторонняя привязка** – синхронизация данных между элементами (TextBox, Slider, CheckBox)
- **Одноразовая привязка** – пример привязки, которая обновляется только один раз при загрузке
- **Односторонние привязки** – отображение данных без возможности редактирования (ProgressBar, TextBlock)
- **Триггеры** – демонстрация PropertyTrigger, DataTrigger, EventTrigger, MultiTrigger

### Управление товаром

В приложении реализована модель товара с полями:
- Название
- Цена
- Количество
- Итоговая сумма (вычисляемое поле)

### Демонстрируемые технологии

- Паттерн MVVM
- Типы привязок: Default, TwoWay, OneWay, OneTime
- Триггеры: PropertyTrigger, DataTrigger, EventTrigger, MultiTrigger
- Два подхода к реализации ViewModel: ручной и с CommunityToolkit.MVVM

## Лабораторная работа №2: Локализация

### Описание

Приложение модифицировано для поддержки двух языков (русский и английский). Реализовано три различных подхода к локализации в отдельных ветках:

- **localization-resx** – использование файлов ресурсов (RESX)
- **localization-dictionary** – использование словарей ресурсов XAML (ResourceDictionary)
- **localization-external** – вынос логики локализации в отдельную библиотеку классов

### Возможности

- Динамическое переключение языка без перезапуска приложения
- Перевод всех статических текстов интерфейса
- Перевод динамических сообщений (MessageBox)
- Переключение языка через выпадающий список

---

## Ветки репозитория

- **master** – ручная реализация ViewModel (INotifyPropertyChanged)
- **with-toolkit** – реализация с CommunityToolkit.MVVM
- **localization-resx** – локализация через RESX-файлы
- **localization-dictionary** – локализация через XAML ResourceDictionary
- **localization-external** – локализация через внешнюю библиотеку классов
